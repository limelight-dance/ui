# Limelight Frontend
Limelight website

### Requirements
- node.js

### Installing
Create and populate `.env` file
```
cp .env.template .env
```

Install packages (do **NOT** use `npm install`) and start
```
yarn

npm start
```

### Deploying
```sh
# You only need to run the following once
heroku git:remote -a limelight-dance-ui

git push heroku master
```
